# Feature spec: Мобильная версия

## 1. Назначение

Мобильная версия нужна, чтобы сайт хорошо работал на телефонах.

Эта feature опирается на `Spec/1 Functional.md`, раздел "Мобильная версия".

## 2. Общие требования

На mobile:
- все кнопки помещаются в экран;
- кнопки меньше среднего размера, но остаются удобными для нажатия;
- график помещается в экран и не ломает страницу;
- пользователь может открыть график на весь экран;
- меню и разделы удобно переключаются;
- текст не вылезает за пределы блоков;
- важные действия доступны без горизонтального скролла.

## 3. Что обязательно проверить

Особенно важно проверить:
- раздел "Мини-обучение";
- раздел "Топ монеты";
- раздел "Проверка монет";
- раздел "Спотовая торговля";
- обычный режим графика;
- полноэкранный режим графика.

## 4. Правила

Mobile-версия не должна ломать главные сценарии сайта.

Пользователь должен иметь возможность:
- проходить мини-обучение;
- смотреть топ монеты;
- проверять монету;
- смотреть график;
- открывать график на весь экран.

## 5. Что проверить

Нужно проверить:
- нет горизонтального скролла;
- кнопки помещаются в экран;
- кнопки удобно нажимать;
- текст не выходит за пределы блоков;
- график помещается в экран;
- график не ломает страницу;
- полноэкранный режим графика открывается;
- полноэкранный режим графика закрывается;
- меню удобно открывается и закрывается.
