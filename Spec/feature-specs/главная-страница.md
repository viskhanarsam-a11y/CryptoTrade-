# Feature spec: Главная страница

## 1. Назначение

Главная страница нужна, чтобы пользователь сразу понял, что такое CryptoTrade, и мог перейти в основные разделы сайта.

Эта feature опирается на `Spec/1 Functional.md`, разделы:
- "Назначение функциональной спеки";
- "Основные разделы сайта";
- "Главные пользовательские сценарии".

## 2. Что видит пользователь

Пользователь должен видеть:
- название сайта CryptoTrade;
- короткое объяснение, что сайт помогает изучать трейдинг, смотреть графики монет и проверять криптоактивы;
- основные переходы к разделам сайта;
- возможность открыть регистрацию или вход;
- возможность переключить язык.

## 3. Основные действия пользователя

Пользователь может:
- перейти в мини-обучение;
- открыть топ монеты;
- открыть проверку монет;
- открыть спотовую торговлю;
- открыть регистрацию или вход;
- переключить язык.

## 4. Правила

Главная страница должна быть простой.

Пользователь не должен долго искать главные действия.

Главная страница должна работать на desktop и mobile.

## 5. Что проверить

Нужно проверить:
- все основные разделы доступны с главной страницы;
- переходы ведут в правильные разделы;
- текст не выходит за границы блоков;
- на mobile нет горизонтального скролла;
- кнопки удобно нажимать на mobile.
