# Feature spec: Топ монеты

## 1. Назначение

Раздел "Топ монеты" нужен, чтобы пользователь мог увидеть список из 20 монет и открыть график выбранной монеты.

Эта feature опирается на `Spec/1 Functional.md`, раздел "Топ монеты".

## 2. Что видит пользователь

Пользователь видит список из 20 монет.

Для каждой монеты нужно показывать:
- название монеты;
- тикер;
- логотип монеты, если он доступен;
- текущую цену, если данные доступны;
- краткое изменение цены;
- кнопку или строку для открытия монеты.

## 4. Основной сценарий

Пользователь:
- открывает раздел "Топ монеты";
- смотрит список монет;
- нажимает на монету;
- открывает экран с графиком этой монеты.

## 5. График монеты

График должен быть похож на TradingView.

В графике должны быть:
- большая рабочая область;
- свечи или линия цены;
- удобный масштаб;
- нормальная читаемость на desktop и mobile;
- возможность открыть график на весь экран.

## 6. Правила

Список топ-20 монет задан в разделе "Список монет для MVP".

Для каждой монеты нужно показывать официальный или проверенный источником логотип, если он доступен.

Если логотип монеты недоступен или не загрузился, сайт должен показать нейтральную заглушку с тикером монеты.

Если текущая цена недоступна, строка монеты не должна ломаться.

Если изменение цены недоступно, сайт должен показать нейтральное состояние или не показывать этот показатель.

## 7. Что проверить

Нужно проверить:
- отображается список из 20 монет;
- у каждой монеты есть название;
- у каждой монеты есть тикер;
- у каждой монеты есть логотип или заглушка с тикером;
- монета открывается по нажатию;
- график выбранной монеты отображается;
- график можно открыть на весь экран;
- раздел работает на mobile.
