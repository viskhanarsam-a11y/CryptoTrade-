# Видео по темам обучения

Эта папка хранит ссылки на видео для раздела "Основы Трейдинга".

Видео разложены по темам из спеки:
- `ордер-блоки`;
- `синхронизация таймфреймов`;
- `слом структуры BOS`;
- `имбаланс`;
- `ликвидность`.

Если для темы видео еще не добавлено, в папке темы остается `README.md` без ссылки на ролик.
UI не должен показывать видеоблок для темы, если в ее папке нет добавленного видео.

Для раздела "Основы Трейдинга" нужно подбирать русскоязычные видео.
Видео на другом языке не считается готовым к публикации в уроке для русскоязычной аудитории, если пользователь явно не разрешил исключение.
