# Feature spec: Навигация

## 1. Назначение

Навигация нужна, чтобы пользователь мог удобно переходить между разделами сайта.

Эта feature опирается на `docs/Spec/1 Functional.md`, раздел "Основные разделы сайта".

## 2. Разделы в навигации

В навигации должны быть:
- главная страница;
- Основы Трейдинга;
- топ монеты;
- проверка монет;
- спотовый график;
- регистрация и вход;
- переключение языка.

## 3. Основные действия пользователя

Пользователь может:
- открыть любой раздел сайта;
- вернуться на главную страницу;
- открыть меню на mobile;
- закрыть меню на mobile;
- понять, какой раздел сейчас активен.

## 4. Правила

Навигация должна быть понятной на desktop и mobile.

На mobile меню должно помещаться в экран.

Меню не должно закрывать важный контент так, чтобы пользователь не мог его закрыть.

## 5. Что проверить

Нужно проверить:
- каждый пункт меню открывает правильный раздел;
- активный раздел понятно выделен;
- mobile-меню открывается;
- mobile-меню закрывается;
- навигация не создает горизонтальный скролл.
