# Feature spec: Основы Трейдинга

## 1. Назначение

Раздел "Основы Трейдинга" нужен, чтобы пользователь мог изучать базовые темы трейдинга внутри сайта.

Эта feature опирается на `docs/Spec/1 Functional.md`, раздел "Основы Трейдинга".

## 2. Темы обучения

В разделе должны быть темы:
- ордер-блоки;
- синхронизация таймфреймов;
- слом структуры BOS;
- имбаланс;
- ликвидность.

## 3. Что видит пользователь в списке тем

Для каждой темы нужно показывать:
- название темы;
- короткое описание;
- обучающий материал;
- видео, если оно добавлено в проект;
- статус прохождения, если пользователь зарегистрирован.

## 4. Что видит пользователь в уроке

Урок может содержать:
- текстовое объяснение;
- русскоязычное видео, если видео добавлено;
- примеры на графике;
- короткий итог по теме.

## 5. Основной сценарий

Пользователь:
- открывает раздел "Основы Трейдинга";
- видит список тем;
- выбирает тему;
- открывает урок;
- смотрит материал или видео;
- возвращается к списку уроков.

## 6. Правила

Раздел "Основы Трейдинга" должен быть доступен без обязательной регистрации.

Статус прохождения показывается только зарегистрированному пользователю.

Если видео не добавлено, сайт не должен показывать сломанный видеоблок.

Для раздела "Основы Трейдинга" нужно подбирать русскоязычные видео, чтобы пользователь понимал материал без дополнительного перевода.

## 7. Что проверить

Нужно проверить:
- все темы из functional есть в разделе;
- каждая тема открывает свой урок;
- урок содержит понятный материал;
- видео отображается только если оно добавлено;
- опубликованные видео для уроков русскоязычные;
- статус прохождения видит только зарегистрированный пользователь;
- раздел удобно работает на mobile.
