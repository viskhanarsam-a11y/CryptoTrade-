# Feature spec: Переключение языка

## 1. Назначение

Переключение языка нужно, чтобы пользователь мог пользоваться сайтом на русском или английском языке.

Эта feature опирается на `docs/Spec/1 Functional.md`, раздел "Переключение языка".

## 2. Доступные языки

Сайт должен поддерживать:
- русский;
- английский.

## 3. Что должно переводиться

Переключение языка должно менять:
- тексты интерфейса;
- названия разделов;
- кнопки;
- основные сообщения сайта.

## 4. Основной сценарий

Пользователь:
- открывает переключатель языка;
- выбирает русский или английский язык;
- видит интерфейс на выбранном языке;
- обновляет страницу;
- остается на выбранном языке.

## 5. Правила сохранения

Если пользователь зарегистрирован, выбранный язык должен сохраняться для пользователя.

Если пользователь не зарегистрирован, выбранный язык можно хранить локально в браузере.

## 6. Что проверить

Нужно проверить:
- русский язык включается;
- английский язык включается;
- основные тексты меняются;
- названия разделов меняются;
- кнопки меняются;
- язык сохраняется после обновления страницы;
- язык не сбрасывается у зарегистрированного пользователя.
