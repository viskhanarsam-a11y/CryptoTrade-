# Дизайн-спеки

Эта папка хранит требования к общему дизайну платформы CryptoTrade.

Здесь нужно описывать не функциональность, а визуальные и интерфейсные правила:

- общий стиль платформы;
- цветовую систему;
- типографику;
- сетку и отступы;
- структуру рабочего пространства;
- внешний вид навигации;
- внешний вид карточек, таблиц, графиков и блоков обучения;
- состояния компонентов;
- фоновые визуальные эффекты;
- правила адаптации под мобильные экраны.

Дизайн-спеки должны дополнять функциональные и технические спеки, но не заменять их.

Если дизайн-решение меняет поведение фичи, нужно также обновить соответствующую feature-спеку.

## Список дизайн-спек

- `анимированный-фон.md` — живой canvas-фон со свечным графиком за основным контентом.
